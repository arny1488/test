◆ lookupTimeZonesForNumberKey()

lookupTimeZonesForNumberKey ( $key)
protected

Returns the list of time zones key corresponds to.

key could be the calling country code and the full significant number of a certain number, or it could be just a phone-number prefix. For example, the full number 16502530000 (from the phone-number +1 650 253 0000) is a valid input. Also, any of its prefixes, such as 16502, is also valid.

Parameters
$keyint the key to look up
Returns
array the list of corresponding time zones
