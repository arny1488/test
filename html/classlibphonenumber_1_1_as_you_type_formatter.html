Detailed Description

Class AsYouTypeFormatter A formatter which formats phone numbers as they are entered.

An AsYouTypeFormatter instance can be created by invoking PhoneNumberUtil::getAsYouTypeFormatter(). After that, digits can be added by invoking inputDigit() on the formatter instance, and the partially formatted phone number will be returned each time a digit is added. clear() can be invokved before formatting a new number.
