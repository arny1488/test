◆ getTimeZonesForGeographicalNumber()

getTimeZonesForGeographicalNumber ( PhoneNumber $number)

Returns a list of time zones to which a phone number belongs.

This method assumes the validity of the number passed in has already been checked, and that the number is geo-localizable. We consider fixed-line and mobile numbers possible candidates for geo-localization.

Parameters
$numberPhoneNumber a valid phone number for which we want to get the time zones to which it belongs
Returns
array a list of the corresponding time zones or a single element list with the default unknown time zone if no other time zone was found or if the number was invalid
