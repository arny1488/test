Find a map of replacements for the address.

If this plugin was provided with a delegate instance of {\ttfamily \doxylink{interface_swift___plugins___decorator___replacements}{Swift\+\_\+\+Plugins\+\_\+\+Decorator\+\_\+\+Replacements}} then the call will be delegated to it.
