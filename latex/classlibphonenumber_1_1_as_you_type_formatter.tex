\doxysubsection{Detailed Description}
Class As\+You\+Type\+Formatter A formatter which formats phone numbers as they are entered.

An As\+You\+Type\+Formatter instance can be created by invoking Phone\+Number\+Util\+::get\+As\+You\+Type\+Formatter(). After that, digits can be added by invoking input\+Digit() on the formatter instance, and the partially formatted phone number will be returned each time a digit is added. clear() can be invokved before formatting a new number.
