Loads an image from a file, URL, HTML input file field, binary string, or a valid image handle. The image format is auto-\/detected.

Currently supported formats\+: PNG, GIF, JPG, BMP, TGA, GD, GD2, WEBP.

This function analyzes the input and decides whether to use \doxylink{class_wide_image_1_1_wide_image_a153086e4c8080ed4a5b6c59b2db429e3}{Wide\+Image\textbackslash{}load\+From\+Handle()}, \doxylink{class_wide_image_1_1_wide_image_a934ca4f5e6fa1c361fc699e55ca9bbc9}{Wide\+Image\textbackslash{}load\+From\+File()}, \doxylink{class_wide_image_1_1_wide_image_a759ee52e4ce588dd34975d34587f7c03}{Wide\+Image\textbackslash{}load\+From\+Upload()} or \doxylink{class_wide_image_1_1_wide_image_a9d1b84a614b4aabb45235c914d44e141}{Wide\+Image\textbackslash{}load\+From\+String()}, all of which you can also call directly to spare \doxylink{namespace_wide_image}{Wide\+Image} some guessing.

\doxylink{class_arrays}{Arrays} are supported for upload fields; it returns an array of loaded images. To load only a single image from an array field, use \doxylink{class_wide_image_1_1_wide_image_a759ee52e4ce588dd34975d34587f7c03}{Wide\+Image\textbackslash{}load\+From\+Upload}(\textquotesingle{}img\textquotesingle{}, \$i), where \$i is the index of the image you want to load.

{\bfseries{Note\+:}} the resulting image object takes ownership of the passed handle.
