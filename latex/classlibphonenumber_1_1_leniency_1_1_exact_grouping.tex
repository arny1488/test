Phone numbers accepted are Phone\+Number\+Util\+::is\+Valid\+Number() valid and are grouped in the same way that we would have formatted it, or as a single block. For example, a US number written as "{}650 2530000"{} is not accepted at this leniency level, whereas "{}650 253 0000"{} or "{}6502530000"{} are. Numbers with more than one \textquotesingle{}/\textquotesingle{} symbol are also dropped at this level.

Warning\+: This level might result in lower coverage especially for regions outside of country code "{}+1"{}. If you are not sure about which level to use, email the discussion group \href{mailto:libphonenumber-discuss@googlegroups.com}{\texttt{ libphonenumber-\/discuss@googlegroups.\+com}}.
