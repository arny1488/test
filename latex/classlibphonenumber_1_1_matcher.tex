\doxysubsection{Detailed Description}
Matcher for various regex matching

Note that this is NOT the same as google\textquotesingle{}s java Phone\+Number\+Matcher class. This class is a minimal port of java\textquotesingle{}s built-\/in matcher class, whereas Phone\+Number\+Matcher is designed to recognize phone numbers embedded in any text.
